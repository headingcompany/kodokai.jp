WP Zen-Coding is a awesome `Zen-Coding` plugin for WP admin page.

== Description ==

This plugin will not be update anymore.

Install the [WP Emmet](http://wordpress.org/plugins/wp-emmet/) instead and use the Emmet (previously known as Zen Coding).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-zen-coding` directory to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Can use 'Zen Coding' in textarea of admin page(wp-admin)
4. Please change setting from `Zen Coding` of the `Settings` menu if necessary
